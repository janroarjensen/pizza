# Napoliansk Pizza PRO+

Dette er en GitHub Pages-klar versjon av pizza-appen.

## Innhold

- `index.html` – hele appen i én fil
- `README.md` – kort forklaring
- `.nojekyll` – gjør at GitHub Pages publiserer filen direkte
- `.gitignore` – små standard-unntak

## Publisering på GitHub Pages

1. Opprett et nytt repository på GitHub.
2. Last opp `index.html`, `README.md` og `.nojekyll`.
3. Gå til **Settings** → **Pages**.
4. Velg:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Lagre.
6. Etter litt tid er siden tilgjengelig via GitHub Pages-adressen til repoet ditt.

## Lokal bruk

Du kan også åpne `index.html` direkte i nettleseren.
